\# Workout Progress Intelligence System



Workout Progress Intelligence is a rule-based analytics system designed to analyze workout data and detect training plateaus, muscle imbalances, and long-term progression trends.

The goal of the project is to help users understand their fitness performance using simple data-driven insights.

\## Features


* Log workout sessions and exercises
* Analyze strength progression over time
* Detect potential training plateaus
* Identify muscle group imbalances
* Track long-term performance trends


\## Tech Stack

* Java – backend analytics engine
* HTML / JavaScript – frontend interface
* CSV file storage – workout data persistence
